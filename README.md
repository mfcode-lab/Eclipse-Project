# Eclipse Project üåë

An engaging web platform dedicated to solar and lunar eclipses, featuring educational information, interactive games, and resources to explore these celestial phenomena.

## About the Project

Eclipse Project is an interactive platform designed to make learning about eclipses fun and accessible. Users can:
- Explore detailed information on upcoming and historical eclipses
- Play educational games that simulate eclipse mechanics
- Access resources, visualizations, and safety tips for viewing eclipses

This project was developed for the **NASA International Space Apps Challenge 2023** local event in Pittsburgh, PA, under the theme "The Eclipse Fun Station". It is based on the challenge "Eclipses: Perspective is Everything" This challenge requested participants to create an activity, game, or story explaining the mechanics of an eclipse.
